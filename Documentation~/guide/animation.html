Animation & tweens

One composite animation type, a preset library, and a pooled tween engine ticked by a single central service.

UIAnimation — five channels

MoveRotateScaleFadeColor

Each channel has its own TweenSettings: duration, delay, loops, play mode (Normal · PingPong · Spring · Shake), and easing — either one of ~30 built-in Ease curves (Sine/Quad/Cubic/Quart/Quint/Expo/Circ/Back/Elastic/Bounce in In/Out/InOut flavors, plus Linear and Spring) or a custom AnimationCurve. Move supports directional presets (UIMoveDirection: Left, Top, BottomRight, CustomPosition, …) so "slide in from left" is one dropdown.

The preset library

UIAnimationPreset assets store named, reusable animations, addressed — as always — by Category/Name. Any preset asset dropped under the library folder is auto-discovered; no registration needed. The setup wizard installs 40 presets across seven categories:

Applying presets

In the inspector: every animator slot (the "Show" animation on a container animator, the "Hover" animation on a selectable) has a picker grouped by category. Hovering a preset plays a live preview; clicking it copies the animation data into the slot. Presets are copy-on-apply, not linked — editing a preset later never changes widgets that already used it, and you can freely tweak the copy per-widget.

In a spec: reference presets by full name on the matching field:

{ "id": "Menu/Main",
  "showAnimation": "Show/FadeIn",
  "hideAnimation": "Hide/SlideOutLeft" }

In the Design System window: the Motion tab is the library's home — a searchable browser on the left, the selected preset's five-channel editor on the right, with play/scrub preview on a preview stage or any scene RectTransform.

Animator roles

NeoAnimatorRoles names the standard animation slots, and NeoUISettings maps each role to a default preset — so every new widget animates consistently with zero per-widget setup, and re-mapping a role restyles the whole project's motion at once:

View/ShowView/Hide Button/HoverButton/Press Selectable/NormalSelectable/SelectedSelectable/Disabled Toggle/OnToggle/Off LoopOneShot

Each role suggests matching preset categories first in the picker (View/Show surfaces the Show category, and so on).

The tween engine

Tweens are pooled (TweenPool) and ticked by UITick, a single central service using unscaled delta time — no per-component Update(). The runtime FPS cap in settings throttles it if needed.

Invariant

Tween lifetime. Bind tweens to their targets with tween.SetTarget(owner, get, set) so a destroyed target self-stops the tween instead of throwing every frame. UITick logs-and-drops any tickable that throws — loudly, once.
