# Changelog

All notable changes to Neo UI Framework are documented here.
Versioning follows semantic versioning.

## [0.9.0] — 2026-07-12

First release candidate.

### Added
- Widget set: buttons, toggles, switches, tabs, sliders, steppers, dropdowns,
  inputs, progress (linear/radial), lists, counters, badges, popups, tooltips.
- Containers and layout: views, panels, overlays, stacks, grids, scroll,
  safe-area; responsive roots.
- NeoShape SDF vector graphics — all visuals batch on one shared material;
  Tier-1 batched effects and Tier-2 shader variants; particle emitter.
- Animation system with edit-mode preview and preset library.
- Flow graph navigation with runtime debugging (active-node pulse).
- Theming: color tokens, named variants, live edit-mode updates.
- Signals: category/name event bus with code-first API.
- Settings and cheat menus: catalog-driven, pluggable persistence, input
  rebinding, code bindings, preview/commit slider signals.
- Agent-first authoring: JSON spec format, generator, exporter, three-way
  merge sync protocol, drift detection, Agent Bridge, binding stub generation.
- Native authoring: palette, presets, ID pickers, setup/reset wizards.
- 12 showcases with isolated generated roots.
